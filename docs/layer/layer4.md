# Layer 04: Market Intelligence (10% Weight)

Market Intelligence focuses on the "behavioral authenticity" of the token's trading activity. It distinguishes between genuine investor interest and artificial volume (wash trading) created by the deployer to entice "FOMO" from retail traders.

---

## 1. Screening Methodology

This layer analyzes the transaction stream to find circular and non-organic patterns:

1. **Volume-to-Organic Ratio:** Comparing total trading volume against the number of unique active wallets.
2. **Circular Flow Detection:** Identifying "Loop" transactions where funds move between a small set of wallets to inflate volume.
3. **Tax Configuration Audit:** Checking for hidden "Buy/Sell Tax" traps that prevent investors from exiting.

## 3. Logical Screening Scenarios

- **The "Volume Miracle":** A coin has $5,000,000 in volume within 1 hour, but only has 12 unique holders. This is 100% automated wash trading.
- **The Sell Trap:** The contract is configured with a 99% sell tax. Investors "buy the dip" but their tokens are effectively worth zero because they cannot be sold.
- **Ping-Pong Trading:** Miner detects that 80% of volume is coming from only 3 wallets trading among themselves.

## 4. Technical Requirements for Miners

Miners in this layer focus on **Transaction Stream Analysis**.

### A. DexScreener / Birdeye / Defined API

- **Usage:** Miners need high-speed access to aggregated DEX data to monitor volume and wallet counts.
- **API Requirement:** Pro/Enterprise API keys are often needed to get sub-minute updates on volume spikes.

### B. Heuristic Pattern Matching

- **Logic:** Algorithms that track the "Life of a SOL" to see if it endlessly returns to the same 3-4 wallets.
- **Tooling:** Python `pandas` for time-series analysis of transaction density.

### C. Simulation Engines

- **Usage:** Miners may run "Transaction Simulations" to test if a 'Sell' order would actually succeed on-chain given the current contract state.

## 5. Scoring Formula (Layer 04)

`L4_Score = (Wash_Trading_Penalty * 0.4) + (Slippage_Trap_Penalty * 0.4) + (Circular_Flow_Penalty * 0.2)`

**WARNING for Miners:** Be careful of "MEV Arbitrage" which can sometimes look like wash trading. Advanced miners distinguish between profitable arbitrage and malicious wash trading designed to fake organic growth.
